# Mina Backend

Welcome to the Mina Backend repository! This project provides the backend functionality for the Mina application, including database integration, Redis caching, and SQLite management.

## Project Structure

- **Backend:** The main backend is hosted on [Mina Backend](https://mina-backend-1.onrender.com).
- **Redis Viewer:** Use this viewer to monitor Redis data: [Mina Redis Viewer](https://sanjeevrae.github.io/mina-redis/).
- **SQLite Viewer:** The SQLite data viewer for inspecting SQLite database : [Mina SQLite Viewer](https://sanjeevrae.github.io/mina-SQLite/).
- 
## Features

- **Database Connection:** A robust and secure database connection to manage the application's data.
- **Caching with Redis:** Optimizes performance by caching frequently accessed data in Redis.
- **SQLite Management:** Provides a simple SQLite database setup for local development and testing.

## Setup Instructions

To get started with this repository, follow these instructions:

### Prerequisites

Ensure you have the following installed:

- Node.js (version >= 14)
- npm (version >= 6)
- Redis (if you plan to use local Redis instance)
- SQLite (for local database viewing and management)
